# フラッシュ暗算ゲーム

ツクリテイデアイベント用のフラッシュ暗算ゲームです。

## 特徴

- 20級から10段まで対応した本格的なフラッシュ暗算
- レスポンシブデザインでスマートフォンにも対応
- マニュアル設定で自由にカスタマイズ可能
- 美しいアニメーションと効果音

## 使用方法

1. レベルを選択（級位・段位・マニュアル設定から選択可能）
2. STARTボタンを押してゲーム開始
3. 数字が順番に表示されるので暗算で計算
4. 答えを入力して判定ボタンを押す

## レベル説明

### 級位（20級〜1級）
- 20級: 1桁 3口 5秒（初心者向け）
- 10級: 2桁 2口 4秒（デフォルト）
- 1級: 2桁 15口 13秒（上級者向け）

### 段位（初段〜十段）
- 初段: 3桁 4口 4秒
- 十段: 3桁 15口 3秒（最高レベル）

### マニュアル設定
- 桁数: 1〜4桁
- 口数: 2〜15口
- 速度: 0.2〜2.0秒

## 技術仕様

- HTML5 + CSS3 + JavaScript（ES6+）
- Google Fonts使用
- レスポンシブWebデザイン
- Vercel対応

## デプロイ

このプロジェクトはVercelで簡単にデプロイできます：

1. GitHubにリポジトリを作成
2. ファイルをプッシュ
3. Vercelでリポジトリを連携
4. 自動デプロイ完了
